# ⚡ Scripts de Automatización del Sistema

Esta sección almacena las herramientas de scripting desarrolladas para optimizar tareas operativas, mitigar errores humanos y agilizar el diagnóstico en ventanas de soporte 24/7.

## Contenido:
1. **`network_diagnostic.sh` (Bash):** Diseñado para análisis perimetral de latencia e ICMP multiregión.
2. **`cleanup_ecr.py` (Python):** Automatización de optimización de costos mediante depuración de almacenamiento huérfano en AWS.
3. **`Get-ADHealthCheck.ps1` (PowerShell):** Auditoría y cumplimiento normativo de identidades en entornos Windows Server / Active Directory.

## Modo de uso:
Asegúrese de otorgar permisos de ejecución en entornos Linux/Unix:
```bash
chmod +x network_diagnostic.sh
./network_diagnostic.sh
